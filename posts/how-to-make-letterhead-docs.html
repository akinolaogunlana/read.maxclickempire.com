Step‑by‑Step: Design Your Letterhead

1. Open or create a new Google Doc

2. Enable header/footer

  • Go to Insert → Headers & footers.
  • Select “Header” to add your logo and branding details at the top.
  • Insert “Footer” for address, website, or legal notices at the bottom. 2

3. Insert your logo and basic info

  • In the header, choose Insert → Image → Upload from computer to add your logo. 3
  • Resize and align it using the image toolbar.
  • Below or beside the logo, add your business name, address, phone, email, and website.

4. Add visual accents

  • Use lines or borders: Insert → Drawing → + New → Line/Shape to add clean dividers. 4
  • Consider subtle color accents using your brand palette. Avoid clutter. 5

5. Adjust layout and spacing

  • Go to File → Page setup to customize margins—reduce top margin if needed. 6
  • Ensure enough white space around header/footer for a balanced look. 7

6. Choose professional fonts and alignment

  • Use fonts like Arial, Calibri, or Times New Roman for readability. 8
  • Stick to 10–12 pt font size for contact details—avoid overly stylized fonts.
  • Align text consistently—left or centered based on your design.

7. Preview before finalizing

  • Use File → Print preview to check positioning and spacing. 9
  • Make adjustments as needed to ensure print and PDF layout is perfect.

8. Save it as a template

  • Option A: File → Make a copy and rename it “Letterhead template”.
  • Option B (for Google Workspace users): Submit your design via Template Gallery → Your organization → Submit template. 10

Pro Tips for a Professional Finish

  • Consistency is key: Use the same font and colors across all materials.

  • Test output: Print to PDF and check alignment on paper.
