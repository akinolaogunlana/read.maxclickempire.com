How to Use Chicago Citation Style in Google Docs (Step‑by‑Step Guide)

Crafting a paper in Chicago style using Google Docs should feel straightforward—even if you're new to the format or platform. This human‑centered walk‑through guides you through every essential element: clean formatting, precise footnotes, and a properly structured bibliography.

1. Set Up Document Formatting

Begin by configuring your document to meet Chicago style standards:

  • Margins: 1‑inch on all sides
  • Font: Times New Roman (or similar), 12 pt
  • Spacing: Double‑spaced text; no extra spacing between paragraphs
  • Paragraph indent: First line indented by 0.5″ using ruler or Format → Align & Indent → Indentation options
  • Page numbers: Insert in header, upper right. For files with a title page, start numbering on page 2 as “2”

2. Create a Title Page

Chicago style title page is simple but precise:

  • Center text about one third down: your paper’s title (in regular text, not bold)
  • Two thirds down: your name, course (or affiliation), instructor’s name, and date—each on a new line, single‑spaced or as required

Press Enter keys to achieve vertical placement. Do not include page number on the title page.

3. Insert Footnotes for Citations

Chicago “notes and bibliography” uses footnotes (or endnotes). In Google Docs:

  • Place cursor directly after the quoted or paraphrased text
  • Use Insert → Footnote (or Ctrl+Alt+F) to insert a superscript number and footnote area
  • In the footnote: use font size 10, single‑spaced, with a first‑line indent
  • Subsequent citations from the same source can be shortened (author’s last name, short title, page number)

4. Build a Chicago Bibliography

At the end of your document, add a bibliography page:

  • Insert a page break
  • Center the heading “Bibliography” or “Works Cited” as required
  • Enter each entry in Times New Roman 12 pt, single‑spaced with a hanging indent (0.5″ indent for all lines except the first) via Format → Align & Indent → Indentation options
  • Leave one line space between each entry

5. Use Google Docs Citations Tool (Chicago Author‑Date)

Google Docs offers a built‑in Citations sidebar (Tools → Citations):

  • Select “Chicago Author‑Date (17th ed.)” style
  • Add citation sources by ISBN or URL, or manually input sources
  • Insert in-text citations—while manually replacing “#” with page numbers if needed
  • Insert bibliography automatically at the end of your document

Note: This tool supports Chicago Author‑Date style only, not Notes‑Bibliography style.

6. Additional Formatting Tips

  • Enable the ruler (View → Show ruler) to adjust indents visually
  • Use the outline pane with header styles to navigate long documents
  • Share your document with collaborators or reviewers and use comments to capture feedback

These productivity hacks make drafting, editing, and reviewing smoother and more transparent.

Frequently Asked Questions

Q: Which Chicago style format does Google Docs support?

A: Google Docs natively supports the Chicago Author‑Date (17th edition) style through its Citations tool. The Notes‑Bibliography format must be managed manually using footnotes and bibliography entries .

Q: How do I format subsequent citations of the same source?

A: After the initial full citation in a footnote, use a shortened note: author’s last name, short title, and page number. Consecutive citations can also use "ibid." where applicable .

Q: Can I switch styles mid-document?

A: It's best to choose your citation format early. Mixing Author‑Date and Notes styles in one document is discouraged. If needed, restart the bibliography part or use separate sections.

Q: How do I ensure hanging indents for bibliography entries?

A: Highlight the bibliography entries, then go to Format → Align & Indent → Indentation options → set “Special indent: Hanging” to 0.5″, and click Apply .

Q: What font and spacing settings are required?

A: Use a readable serif font (like Times New Roman), 12‑point size, double‑spaced body text, and size‑adjusted footnotes (10 pt). Paragraphs should be first‑line indented 0.5″. All margins should be one inch .

Conclusion

This guide presents a full Chicago style layout workflow in Google Docs—covering document setup, title page design, footnotes, bibliography, and citation tool options. Whether using Notes‑Bibliography or Author‑Date, you'll gain clarity, structure, and academic credibility.
